Servizi

Web Development

Ho studiato lo sviluppo di Siti Web Responsive, Content First e Mobile First con i linguaggi HTML e CSS, nello specifico nella versione HTML5 e CSS3. Ho affrontato il linguaggio JavaScript con i framework jQuery, jQuery UI, per l'interattività dell'utente con il web e Pre-processori CSS come SASS. Infine utilizzo la suite di Seotester per controllare la seo del sito, in modo da scoprire l'efficacia che il sito ha su Google.

Web Design

L'ambiente di sviluppo che utilizzo maggiormente è Visual Studio Code.
I software per gli elementi grafici e di prototipazione sono Figma, Adobe XD mentre per l’ editing delle immagini utilizzo Photoshop.
Per reperire Font, Icone, Immagini, utilizzo Font Awesome, Font Squirrel, Pixabay, Shutterstock e le altre numerose raccolte presenti nel Web, sia gratuite che a pagamento.

Manutenzione

Non basta creare un sito web, bisogna anche tenerlo aggiornato (es. inserimento o modifica di funzioni e plugin, servizi di backup delle pagine e del database, servizi statistici…) si può incorrere in singole problematiche o nuove opportunità (es. interventi legati a problematiche dei server e dell’hosting, evoluzione delle logiche di funzionamento dei motori di ricerca, nuovi sviluppi e funzionalità del software …).
